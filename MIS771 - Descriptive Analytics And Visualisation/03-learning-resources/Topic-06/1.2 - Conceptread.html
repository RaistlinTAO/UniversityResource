Key concepts

The multiple regression model describes the relationship between the dependent variable (Y), a set of independent variables (xâ€™s) and an error term (e). To build a multiple regression model the basic three aspects are important.

  1. Model Specification: Decide what you want to do and select the dependent variable, determine the potential independent variables for your model, gather sample data (observations) for all variables.
  2. Model Building: The process of constructing a mathematical equation that explains the relationship between independent variables and the dependent variable. The model building process involved the following steps:
    • identification of potential independent variables (using scatter diagram and correlation matrix),  
    • creation of regression model with chosen independent variables,
    • checking for the significance of the model and for each variable,
    • exclusion/inclusion of independent variable,
    • residual analysis and obtaining the final model.
  3. Model Diagnosis: Under model diagnosis, the following questions need to investigate:
    • Is the overall model significant?
    • Are the individual variables significant?
    • Is the standard deviation of the model error too large to provide meaningful results?
    • Have the regression analysis assumptions been satisfied?
