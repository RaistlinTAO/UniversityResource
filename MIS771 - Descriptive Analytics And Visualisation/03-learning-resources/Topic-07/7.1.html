Logistic Regression 

We talked about multiple regression and some advanced topics in regression analysis before the break. We also learned how to use categorical variables (by dummy coding them) to predict continuous outcomes.

However, we have not yet looked at the reverse process: predicting categorical outcomes from continuous or categorical predictors.

In a nutshell, logistic regression is multiple regression but with a categorical outcome variable and predictor variables that are categorical or continuous. A trivial example is to determine which variables predict whether a student passes or fails the final exam (a binary categorical variable). We might use the number of social engagements, attendance, performance in other assessments, etc., as predictors.

Logistic regression could help us identify which variables predict pass or fail in the exam and predict whether a new student is likely to pass or fail the exam based on these variables. In this topic, we extend our understanding of regression analysis by discussing Logistic Regression and its applications.

At the completion of this topic, you will be able to learn

  • Circumstances where logistic regression is the appropriate application of regression analysis instead of multiple regression model.  
  • Identify the types of dependent and independent variables used in the application of logistic regression
  • Analyse and interpret the output a logistic regression analysis and assessing predictive accuracy
  • Understand the strengths and weaknesses of logistic regression compared to multiple regression.
  • Apply logistic regression analysis to business decision-making situations
